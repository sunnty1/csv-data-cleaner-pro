\# CSV Data Cleaner Pro



Simple Python tool to clean and organize CSV files automatically.



\## Features



\* Remove duplicate rows

\* Fill missing values

\* Format names and text

\* Generate cleaning reports

\* Custom CSV file input

\* Export cleaned CSV



\## Requirements



Install dependencies:



```bash

pip install pandas

```



\## Run



```bash

python main.py

```



Enter your CSV file name:



```text

dirty\_customers.csv

```



Output:



```text

cleaned\_customers.csv

report.txt

```
